# Frontend (React + Vite)

This folder contains the React frontend for the Instagram Automation backend.

## Features

- Login/logout with backend token authentication
- Engagement score request UI
- Followers export UI with download link
- Post/reel download UI
- Profile picture download UI

## Setup

1. Install dependencies:

```bash
npm install
```

2. Configure frontend API URL:

```bash
cp .env.example .env
```

3. Run development server:

```bash
npm run dev
```

The app runs on `http://localhost:5173` by default.

## Build

```bash
npm run build
```
