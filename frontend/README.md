# Frontend (React + Vite)

This folder contains the dashboard UI for the Instagram Automation backend.

## Features

- Authentication with backend session token
- Follower analysis views (followers, unfollowers, not-following-back, mutual)
- Summary statistics panel
- Post and reel download panel with direct media file links

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

Default URL:

- http://localhost:5173

## Build

```bash
npm run build
```
